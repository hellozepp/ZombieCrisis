## 🔧 快速添加注释指南

### 注释模板

```java
/**
 * 类名 - 简短描述
 * 继承关系说明
 * 主要功能说明
 */
public class ClassName extends ParentClass {
    
    /**
     * 属性说明
     */
    private int attribute;
    
    /**
     * 方法说明
     * @param param 参数说明
     * @return 返回值说明
     */
    public int method(int param) {
        // 关键代码注释
        return result;
    }
}
```

### 注释要点
1. **类注释**: 说明类的作用、继承关系、主要功能
2. **方法注释**: 说明方法用途、参数含义、返回值
3. **关键代码**: 复杂算法、重要逻辑添加行内注释
4. **常量**: 说明常量的含义和用途
5. **内部类**: 说明内部类的作用
